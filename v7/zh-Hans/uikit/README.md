## TinaX UI套件

TinaX.UIKit模块是TinaX框架的UI业务功能基础框架模块。

但UIKit模块是一个纯逻辑框架，并没有包含具体UI系统的实现，因此通常我们需要配合UIKit的一个内容提供模块来使用，如使用 uGUI (Unity UI) 系统的 [TinaX.UIKit.UGUI](/zh-Hans/uikit/ugui/README)模块。
